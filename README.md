🏠🔒 집보장
2030 세대를 위한 전세 사기 위험 진단 서비스

집보장(Jipbojang) 은 전세 계약 과정에서 발생할 수 있는 위험 요소를 사전에 파악하고,
임차인이 스스로 데이터를 기반으로 안전한 결정을 내릴 수 있도록 돕는 전세 사기 예방 플랫폼입니다.

📌 프로젝트 개요

전세 사기 피해는 최근 몇 년간 급증했으며, 특히 20~30대 청년층의 피해가 두드러지고 있습니다.
집보장은 복잡한 부동산 서류와 정보 비대칭 문제를 해결하기 위해
전세 위험 진단·등기 변동 알림·전문가 상담·부동산 챗봇·커뮤니티 기능을 제공하여
임차인이 스스로 매물을 평가하고 위험을 줄일 수 있도록 돕습니다.
